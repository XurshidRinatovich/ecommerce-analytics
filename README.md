# 🛒 E-Commerce Sales Analytics | UK Online Retail 2010–2011

Полный аналитический проект на основе реального датасета британского интернет-магазина.  
Стек: **PostgreSQL · Python · Power BI**

---

## 📊 О проекте

Анализ 541 909 транзакций розничного магазина за 2010–2011 год.  
Цель — выявить закономерности в продажах, сегментировать клиентов по методологии RFM и построить интерактивный дашборд.

| Показатель | Значение |
|---|---|
| Исходных строк | 541 909 |
| После очистки | 397 880 |
| Уникальных клиентов | 4 338 |
| Уникальных заказов | 18 532 |
| Общая выручка | $8 911 407 |
| Стран | 38 |

## 🚀 Этапы работы

### 1. SQL: загрузка и очистка данных

Данные загружены в PostgreSQL (541 909 строк). Создан VIEW `clean_orders` с фильтрацией:
- Отменённые заказы (invoice_no начинается на "C") — удалены
- Отрицательные количества (возвраты) — удалены
- Нулевые/отрицательные цены — удалены
- Строки без customer_id — удалены

Итого после очистки: **397 880 строк** (73% от исходного датасета).

### 2. SQL: RFM-анализ

Для 4 338 уникальных клиентов рассчитаны три метрики:
- **R (Recency)** — дней с последней покупки
- **F (Frequency)** — количество уникальных заказов
- **M (Monetary)** — общая сумма покупок

### 3. Python: статистика и визуализация

- Описательная статистика суммы заказов
- Корреляционный анализ с проверкой значимости (scipy.stats.pearsonr)
- Выявление выбросов через Z-score
- 5 графиков сохранены в папку `images/`

### 4. Power BI: интерактивный дашборд

- 3 карточки KPI (выручка, клиенты, заказы)
- График динамики по месяцам
- График выручки по RFM-сегментам
- Топ-10 товаров по выручке
- Срез (фильтр) по стране — интерактивная фильтрация всего дашборда

## 🔍 Ключевые выводы

### 1. Сезонность продаж
Выраженный пик в **ноябре 2011** (~$1.16 млн) — предновогодний шопинг.  
Слабые месяцы — **февраль и апрель** (~$450–470 тыс.) — постпраздничный спад.

### 2. Правило Парето в клиентской базе
**VIP-сегмент** (348 клиентов = 8% базы) приносит **$4.28 млн = 55% всей выручки**.  
"Lost" + "At Risk" — 33% клиентской базы, но всего 13% выручки.

### 3. Сильная корреляция Sales → Profit
Pearson r = **0.914**, p-value ≈ 0 — статистически значимая связь,  
не случайность: чем больше количество товаров в заказе, тем выше итоговая сумма.

### 4. Аномалии в данных
Найдено **150 выбросов** (0.04% данных) — преимущественно крупные оптовые сделки.  
Пример: один заказ на **80 995 штук** товара "PAPER CRAFT, LITTLE BIRDIE" = $168 469.  
Такие сделки искажают средние значения и топ-листы — требуют отдельного учёта.

### 5. География продаж
91% выручки — **United Kingdom** (домашний рынок).  
Среди международных: **Netherlands** ($285 тыс.) лидирует несмотря на всего 94 заказа —  
средний чек ~$3 036 против ~$500 у Германии. Вероятно, оптовые покупатели.
